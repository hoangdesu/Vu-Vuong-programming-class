## Conway's game of life

Run with any of the following:

`./game_of_life`
> 40x40
> 
`./game_of_life 20`
> 20x20
> 
`./game_of_life 20 30`
> 20x30

Controls are as follows:

- play: GREEN CONCRETE
- pause: RED CONCRETE
- dead: BLACK CONCRETE
- alive: WHITE CONCRETE
- speed up: YELLOW CONCRETE
- slow down: BLUE CONCRETE
- randomise: PINK CONCRETE
- clear: BARRIER
